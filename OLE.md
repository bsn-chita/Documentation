# OLE соединение.
OLE-соединение (Object Linking and Embedding) в 1С позволяет управлять одной информационной базой из другой или из внешнего приложения. Сейчас чаще используется более быстрый механизм COM-соединения, но OLE остается актуальным, если нужно запустить полноценное приложение с интерфейсом.

> OLE-соединение можно использовать для перекачки данных между конфигурациями (очень медленно).

1. Создание объекта. В зависимости от версии 1С используются разные идентификаторы. Например, _V83.Application_ запускает «толстый клиент» 1С:Предприятие 8.3.

2. Строка подключения.
Параметры зависят от типа базы:
    - Файловая база: File="ПутьКБазе"; Usr="Логин"; Pwd="Пароль";.
    - Серверная база: Srvr="ИмяСервера"; Ref="ИмяБазы"; Usr="Логин"; Pwd="Пароль";.
